# 업무 칸반 - 기능 설명서

## 1. 칸반 보드 (메인 화면)

### 1-1. 3단 컬럼 구조
- **할 일** (todo) - 새로 생성된 카드가 여기에 배치
- **진행중** (in_progress) - 작업 중인 카드
- **제출완료** (submitted) - 완료된 카드

### 1-2. 드래그 앤 드롭
- 카드를 마우스로 잡아서 다른 컬럼이나 위치로 이동 가능
- 같은 컬럼 내에서 순서 변경도 가능
- @dnd-kit 라이브러리 사용

### 1-3. 카드 정보 표시
- 제목, 우선순위 배지, 라벨 배지
- 마감일 (D-day, D-1, D+n 지남 등 자동 계산)
- 시작/종료 시간
- 메모 미리보기 (2줄 제한)

---

## 2. 카드 관리

### 2-1. 카드 생성/수정 (모달)
- **제목** (필수)
- **우선순위**: 긴급 / 높음 / 보통 / 낮음
- **라벨**: 보고서, 점검, 품의, 가전표, 결재, 교육 이수, 업체 연락처, 회의, 회의자료, 기타
- **마감일**: 날짜 선택
- **시작/종료 시간**: 시간 선택
- **메모**: 자유 텍스트

### 2-2. 체크리스트
- 카드 수정 시 체크리스트 항목 추가/삭제/토글 가능
- Enter 키로 빠르게 항목 추가
- 완료된 항목은 취소선 표시

### 2-3. 첨부파일
- 카드에 파일 첨부 가능 (모든 파일 형식)
- 파일 아이콘 자동 표시 (PDF, Excel, HWP, 이미지, Word 등)
- 첨부파일 클릭 시 해당 파일 열기
- 첨부파일 삭제 가능

### 2-4. 카드 복제
- 기존 카드의 제목, 메모, 우선순위, 마감일, 라벨, 시간 정보를 복사하여 새 카드 생성
- 제목 뒤에 "(복사)" 자동 추가

### 2-5. 아카이브
- 제출완료 상태의 카드를 아카이브로 이동 (보드에서 숨김)
- 아카이브된 카드 목록 조회
- 아카이브에서 복원 (할 일 컬럼으로 되돌림)
- 아카이브된 카드 영구 삭제

---

## 3. 필터/검색 (툴바)

- **텍스트 검색**: 제목, 메모에서 키워드 검색
- **우선순위 필터**: 긴급/높음/보통/낮음 중 선택
- **라벨 필터**: 10개 라벨 중 선택
- **마감일 필터**:
  - 직접 날짜 지정 (해당 날짜까지)
  - 오늘까지 / 이번주 / 다음주까지 프리셋
- **+ 새 카드** 버튼

---

## 4. 반복 업무

- 반복 규칙 등록: 매일 / 매주 (요일 지정) / 매월 (날짜 지정)
- 각 반복 규칙에 제목, 설명, 우선순위, 라벨, 자동 마감일(+N일) 설정
- "오늘 카드 생성" 버튼으로 해당일 반복 카드 일괄 생성
- 같은 날 중복 생성 방지 (last_generated 체크)
- 반복 규칙 활성/비활성 토글
- 반복 규칙 삭제

---

## 5. 통계

- **이번 달 제출**: 현재 월에 제출완료된 카드 수
- **활성 카드**: 아카이브되지 않은 전체 카드 수
- **아카이브**: 아카이브된 카드 수
- **기한 초과**: 마감일이 지났지만 완료되지 않은 카드 수
- **우선순위별 활성 카드**: 긴급/높음/보통/낮음 각각의 카드 수

---

## 6. 타이틀바 (상단 메뉴)

### 6-1. 현재 시간 표시
- "2026년 4월 8일 (화) 14:30" 형식
- 1분마다 자동 갱신

### 6-2. 날씨
- Open-Meteo API를 통한 현재 날씨 표시 (기온, 날씨 상태)
- 10개 주요 도시 선택 가능 (인천, 서울, 부산, 대구, 대전, 광주, 울산, 수원, 성남, 고양)
- 선택한 지역 localStorage에 저장
- 30분마다 자동 갱신

### 6-3. 자동시작 (Tauri 전용 - 웹 전환 시 제거)
- Windows 부팅 시 앱 자동 실행 ON/OFF
- `@tauri-apps/plugin-autostart` 사용

### 6-4. 테마 전환
- 다크 모드 / 라이트 모드 토글
- 선택한 테마 localStorage에 저장

### 6-5. 백업/복원
- **백업**: 전체 카드 데이터를 JSON 파일로 다운로드
- **복원**: JSON 파일을 선택하여 데이터 복원 (기존 데이터 덮어쓰기)

### 6-6. 메뉴 순서 커스터마이징
- 상단 메뉴 버튼들을 드래그로 순서 변경 가능
- 설정 아이콘(⚙) 클릭으로 편집 모드 토글
- 순서 localStorage에 저장

---

## 7. 상태바 (하단)

### 7-1. 카드 현황
- 전체 카드 수 | 할일 · 진행중 · 제출완료 각각의 수

### 7-2. 점심 메뉴 추천
- 날짜 기반으로 매일 다른 메뉴 추천
- 클릭하면 다른 메뉴로 변경
- 메뉴 편집 기능 (추가/삭제/초기화)
- 기본 메뉴 70개+ 내장 (A구역, B구역 맛집 포함)
- 커스텀 메뉴 localStorage에 저장

### 7-3. 오늘 처리 현황
- 오늘 제출완료한 카드 수 + 응원 메시지

### 7-4. 마감 임박 알림
- 마감일이 1일 이내인 카드 수 표시
- 클릭하면 해당 카드 목록 팝업

---

## 8. 알림 (Tauri → 웹 전환)

- 앱 시작 시 마감 관련 알림 발송:
  - 오늘 마감 카드 목록
  - 내일 마감 카드 목록
  - 기한 초과 카드 수
- 기존: Tauri 네이티브 알림 → 웹 전환 시: 브라우저 Web Notifications API

---

## 9. 키보드 단축키

- **N**: 새 카드 생성 모달 열기
- **Esc**: 모달/팝업 닫기
- (input/textarea/select에 포커스 중일 때는 비활성)

---

## 10. 데이터 저장

- 기존: SQLite (Rust 백엔드, `kanban.db` 파일)
- 웹 전환 시: IndexedDB (Dexie.js, 브라우저 내장 DB)
- 테이블: tasks, checklist_items, attachments, recurring_tasks

---

## 웹 전환 시 변경 사항 요약

| 기능 | 기존 (Tauri) | 웹 전환 후 |
|------|-------------|-----------|
| 데이터 저장 | SQLite 파일 | IndexedDB (브라우저) |
| 첨부파일 | 파일 경로 저장 → 로컬 파일 열기 | 파일 Blob 저장 → 다운로드/미리보기 |
| 알림 | 네이티브 알림 | 브라우저 Notification API |
| 날씨 | Rust에서 HTTP 요청 | 브라우저 fetch() 직접 호출 |
| 자동시작 | Windows 시작 프로그램 등록 | **제거** (브라우저에서 불가) |
| 실행 방식 | .exe 데스크톱 앱 | 브라우저에서 index.html 열기 |
